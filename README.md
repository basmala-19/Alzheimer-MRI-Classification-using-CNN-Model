# Alzheimer MRI Classification using CNN Model
## Overview
This project implements a Convolutional Neural Network (CNN) to classify stages of Alzheimer's disease based on MRI scans. The model can predict four stages of cognitive impairment:
- Mild Demented
- Moderate Demented
- Non Demented
- Very Mild Demented
